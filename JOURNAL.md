# 4/25/2026 10 AM - Firmware Dev

_Time spent: 3h_

# Overview
I began designing the firmware by deciding how the system should function. I decided to split the design into it different scetions. They mainly handle things like sensor inputs, alert and output system, bluetooth control etc.
Obviously a loop function would be best for continuously running the system

![Screenshot 2026-04-25 103341](https://stasis.hackclub-assets.com/images/1777109954986-j8ejt3.png)

The readSensors function ensures that the sensors are continuosly measuring the temp and humidity and determine the next action based off of the set conditions.

I made the alert system very easy to understand. The LEDs indicate the system state basically. Red for high temp, blue for low, yellow for low humidty (I didnt add a high humidity state cause high humidity isnt much of a controllable problem), white for low light intensity, and green which shows that everything is noormal.
For things like high add low temperature, The buzzer also gives alerts along with the LEDs to make sure that the abnormal conditions are made aware of.
Then theres the lamp system, which is just a combination of about 3 white LEDs connected to a transistor that will function for small scale lighting. It turns on automatically when low light is detected.

The OLED funtions as an information panel that will display the sensor data, system status and what not through different dashboard pages (Environment data, system status, and events page).
Environmental dashboard for the sensor data like temperature, humidity and light levels
system status for displaying things like the bluetooth status, and system mode
events page for viewing the last few commands and alerts (for record keeping purposes)
The system structured this way should be quite practical and easy to use.

And it also has bluetooth control. for example, when the lamp leds turn on, they would normally not turn back off unless light intensity goes back up, so it can be controlled through remote commands, both for turning off and on, that overides whatever current state its in. The bluetooth also allows for mode and page switching.
