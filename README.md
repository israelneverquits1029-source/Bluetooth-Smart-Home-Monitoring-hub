# Bluetooth-Smart-Home-Monitoring-hub
A smart home hub system that monitors room conditions, displays live data and controls output devices.
I made this project because, well, I wanted to make a useful system that is able to read and monitor environmental conditions and their changes, as well as alerting whoever is using it about those changes so they can be controlled.
